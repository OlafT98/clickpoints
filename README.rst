Installation
------------

.. code-block:: bash
   # create & activate an isolated environment (recommended)
   python3.13 -m venv .venv
   source .venv/bin/activate
   # install ClickPoints from PyPI
   pip install --upgrade pip
   pip install clickpoints        # or: pip install clickpoints[video]
   # --- OR --- from source
   git clone https://github.com/OlafT98/clickpoints.git
   cd clickpoints
   pip install -e .               # installs all required packages
